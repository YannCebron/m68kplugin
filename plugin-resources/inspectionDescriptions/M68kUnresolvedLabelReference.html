Highlights unresolved references to labels.

All macros defining a label (via their first parameter) must be configured explicitly. Use the provided quick fixes to add them from usages in code. 


* APTR macro
   macro APTR
\1 rs.l  1
   endm

* label declaration
   APTR  LN_SUCC         ; use "Find Usages" on 'LN_SUCC' parameter

* usage
   move.l a1,LN_SUCC(a0) ; resolves to label declaration macro call
